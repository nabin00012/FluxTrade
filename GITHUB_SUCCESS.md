## 🔄 Making Future Changes

When you make changes to your code:

### Step 1: Check what changed
```bash
git status
```

### Step 2: Add the changes
```bash
git add .
```

### Step 3: Commit with a message
```bash
git commit -m "What you changed"
```

### Step 4: Push to GitHub
```bash
git push
```
